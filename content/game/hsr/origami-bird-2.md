## 小鳥性能 tier

**注: 主観を大きく含みます。**

★★★★★: 銀狼
★★★★☆: ロビン、開拓、ブートヒル
★★★☆☆: 三月、ホタル、青雀
★★☆☆☆: ジェイド

ゲームシステムの理解、操作速度、運、好みなど、多くの要素に左右されます。あくまで参考程度で。

野良マッチであればどの小鳥でも 1 位を狙うことはできる性能があります。どちらかというと異常に強い相手を引かない方が重要です (30 人対戦ですからね……)。

猛者しかいない大会や参加型なら知りません。

## 各小鳥の使い方

8 種の小鳥の能力と使い方を詳しく見ていきましょう。

ここでは、小鳥の能力を立ち回りの観点から 3 つに分類して解説します。

| 分類               | 小鳥                   |
| ------------------ | ---------------------- |
| スーパー爆弾量産型 | 銀狼、青雀、ブートヒル |
| 連爆型             | ジェイド、開拓、ホタル |
| 平和ボタン型       | ロビン、三月           |

また、小鳥の詳細な性能を「爆発力」「安定性」「簡単さ」の 3 つの評価で記しています。

「爆発力」: 運が良いときのスコアの伸びやすさ。高いほど上位層で勝ち残りやすく、1 位争いに強い。
「安定性」: 運が悪いときのスコアの取りやすさ。高いほど生き残りやすく、第 4・5 ラウンドまでに強い盤面に持ち込みやすい。
「簡単さ」: 操作の簡単さ。高いほど考えることが少なく扱いやすい。

### スーパー爆弾量産型

スーパー爆弾量産型は、その名の通りスーパー爆弾をたくさん生成して全消ししまくるタイプの小鳥です。
スーパー爆弾を縦に重ねて「複製グルー」でコピーする戦略が非常に重要なのが特徴です。

スーパー爆弾を縦に積むのは大変ですが、終盤でループを成立できればほぼ勝ちです。最上位層はスーパー爆弾ループを成立させないとそもそも勝負になりません。ループを作った相手に勝つのはロビンの小鳥以外非常に困難です。

銀狼、青雀、ブートヒル全て違う戦略ですが、いずれも共通して連鎖が起きにくいのを全消し回数でカバーするのが特徴です。操作速度、アイテム判断が比較的重要になってきます。

#### 銀狼の小鳥

![銀狼の小鳥](/img/game/hsr/origami-silverwolf.png)

ブルーベリーを 7 個消すたびに、ランダムなフルーツ 1 個をスーパー爆弾に変える。
スーパー爆弾と隣接するフルーツの一を入れ替えると、それと同じフルーツをすべて消すことができる。

| 総合評価 | ★★★★★ |
| -------- | ----- |
| 爆発力   | ★★★★☆ |
| 安定性   | ★★★★★ |
| 簡単さ   | ★★★★★ |

文句無しの最強です。初心者が使っても安定するし、運に左右されることが少なく、高得点も出しやすい。

連鎖が起きにくく上振れたときの点数はブートヒルやロビンに劣りますが、スーパー爆弾の全消し回数で安定できます。

序盤はとにかくブルーベリーを消しまくってスーパー爆弾を作りましょう。
スーパー爆弾が何個か作れたら「幻想カラーの着色剤」「複製グルー」「リセットキューブ」で縦に積みます。青雀の小鳥に比べてやりやすいです。
スーパー爆弾が左右どちらかの一列に 4 個以上並べられたら、たくさん全消しをしましょう。スーパー爆弾が足りなくなったらアドリブでブルーベリーを消し、「複製グルー」が落ちてくるまで耐えましょう。

スーパー爆弾ループに入りやすく、適当にやっても高得点を狙えます。

#### 青雀の小鳥

![青雀の小鳥](/img/game/hsr/origami-qingque.png)

スーパー爆弾を 2 個消すたびに、フィールド上のランダムなフルーツ 1 個をバブル速達に変える。
隣接するフルーツを消去してバブルを割ると、中のアイテムを入手できる。

| 総合評価 | ★★★☆☆ |
| -------- | ----- |
| 爆発力   | ★★★☆☆ |
| 安定性   | ★★★★★ |
| 簡単さ   | ★☆☆☆☆ |

上位層からの評価は低いですが、個人的には高く評価しています。
アイテム仕様を理解すれば銀狼に迫るほどの点数を稼げるようになります。しかもアイテムの回転率が高いおかげでジリ貧に陥りづらく、安定性は全小鳥中最強です。
しかし、点数が伸び悩みやすいデメリットを持ちます。ループを完成させないと点数が全然伸びないので、序盤はループを早めに構築する運と実力が必要です。

青雀の小鳥の立ち回りは非常に特殊です。
小鳥の能力がフルーツではなくスーパー爆弾で起動するため、序盤はまずスーパー爆弾を 2 個作ることが何よりも大切です。そのためには以下のどちらかを狙います。

- 高速で 3 個消しを続けて盤面を入れ替え、アイテムを落とし、「幻想カラーの着色剤」を使う
- 頑張って 5 個消しを見つける
  前者はアイテムのドロップ運と高速に消し続ける実力が必要で、後者は 5 個消し盤面を引く運と 5 個消し盤面を積極的に作る・探す実力が必要です。どちらを狙うかの状況判断もしたいです。このことから青雀の序盤は非常に難しいです。

スーパー爆弾を盤面に数個作れたら、「幻想カラーの着色剤」「複製グルー」「リセットキューブ」を使って頑張って縦に並べます。「リセットキューブ」で端の列に縦に並べるには運が必要なので、何度もアイテムをドロップさせて粘りましょう。

終盤では左右のどちらかの端に 4 個以上のスーパー爆弾を積み、「複製グルー」でコピーしまくればスーパー爆弾 2 個の全消しを連発できます。こうなればループです。高速で全消ししまくりましょう。
盤面に「複製グルー」がある時はアイテム枠を 1 個空けて全消しし、それ以外の場合は適当にやります。左右の端に積んだスーパー爆弾は手を付けず、他のものでやりくりしてください。またスーパー爆弾は積んだ列の隣、2 列目のものから優先的に使いましょう。

ループに入れば、「リセットキューブ」は使用してはいけません。縦に積んだスーパー爆弾が崩れます。
「幻想カラーの着色剤」は端の列のスーパー爆弾を増やすのでたまに使いますが、端の列に 6 個・7 個溜まっている場合は無視して全消しに集中したほうがよいです。
「熱々爆弾」は使っても盤面に影響がありません。無視して全消しに集中したほうがよいです。端の列以外のスーパー爆弾が尽きた時にスーパー爆弾と正方形爆弾で消すために使うことはあります。臨機応変に。

重要なテクニックとして、盤面に複数のアイテムがある時、次のルールでどのアイテムが先に取得されるかが変わります。

1. フルーツ消しが複数回に分かれている場合、先に消した方の範囲に隣接するアイテムが先に取得される。
2. 同時に消したフルーツの中では、左にあるものが先に取得される。
3. 同時に消したフルーツで同じ列にあるものの中では、下にあるものが先に取得される。
   アイテムが溢れて「複製グルー」が取得できずループに失敗することが青雀の小鳥だと起きやすいですが、この仕様を知っていればどのアイテムが取得されるか判断できます。
   「複製グルー」が溢れそうな場合に先に取ったり、スーパー爆弾が盤面に十分多いときにアイテム枠を消費するか全消しをするかの判断基準になったり、知っていると青雀で戦いやすくなります。

とにかく考えることが多い小鳥です。頑張りましょう。

#### ブートヒルの小鳥

![ブートヒルの小鳥](/img/game/hsr/origami-boothill.png)

モモを 5 個消すたびに、フィールド上のランダムなフルーツ 3 個をボーナスモモに変える。
ボーナスモモを消すと、戦力を 3 獲得できる。

| 総合評価 | ★★★★☆ |
| -------- | ----- |
| 爆発力   | ★★★★★ |
| 安定性   | ★☆☆☆☆ |
| 簡単さ   | ★★☆☆☆ |

モモを増やすことで連爆型のような立ち回りをしつつ、偶発的な 5 個消しでスーパー爆弾を増やせる小鳥です。
前環境で暴れていましたが、今環境では銀狼にトップを譲っています。宝箱が大量に湧くステージが削除されたことが大きいです。

ボーナスモモが発生する位置や降ってくるモモの個数など、非常に運要素が強いです。しかし、全消しをするとモモが大量に生成できるため、スーパー爆弾を作りやすいです。

上位層の評価は高いですが、個人的には非常に苦手なのであまりアドバイスができません。すみません……。

### 連爆型

能力で範囲に爆発を起こせるタイプの小鳥です。
全消しや「お掃除スプレー」で連鎖的に爆発が何度も起こるので、短時間で一気にスコアが伸びやすい特徴があります。
当然「複製グルー」でスーパー爆弾ループをするのも強いですが、銀狼・青雀・ブートヒルよりも安定してループに持ち込めません。おそらく「複製グルー」のドロップ率が低いことが理由です。

連鎖中は操作をゆっくり考えられるので、比較的初心者でも扱いやすく、またコントローラーで操作しにくい人にもおすすめできます。

#### 開拓の小鳥

![開拓の小鳥](/img/game/hsr/origami-trailblazer.png)

キウイを 5 個消すたびに、フィールドのランダムな位置で 2 回、2×2 マスの爆発を起こす。

| 総合評価 | ★★★★☆ |
| -------- | ----- |
| 爆発力   | ★★★★☆ |
| 安定性   | ★★★★☆ |
| 簡単さ   | ★★★★★ |

能力の回転率に優れ、爆発の形状も連爆を引き起こしやすい小鳥です。スーパー爆弾がなくてもキウイを消し続けるだけで高得点を目指せるため、ループを作れない初心者にも扱いやすいです。

とにかくキウイを消すこと、「お掃除スプレー」をキウイに使い続けることが何より重要です。
上位で戦う場合、盤面のキウイ以外のフルーツが残り続けるとキウイが増えないので、定期的にスーパー爆弾を使って全消しをすることも重要です。

最上位層で戦うには多少火力不足なのは否めませんが、爆発の位置が良ければたまに銀狼に勝てる程度はあります。

#### ホタルの小鳥

![ホタルの小鳥](/img/game/hsr/origami-firefly.png)

バナナを 7 個消すたびに、フィールドのランダムな位置で 1 回、十字爆発を起こす。

| 総合評価 | ★★★☆☆ |
| -------- | ----- |
| 爆発力   | ★★★☆☆ |
| 安定性   | ★★★☆☆ |
| 簡単さ   | ★★★★★ |

爆発範囲こそ優秀なものの、回転率は控えめです。スーパー爆弾がなくてもある程度戦えます。しかし、十字爆発は連鎖が起きにくいので、点数は伸び悩みやすいです。

立ち回りは開拓の小鳥とほぼ同じです。
とにかくバナナを消しまくることが大切なので、「お掃除スプレー」をバナナに使いましょう。定期的に全消しをして、盤面のバナナ以外のフルーツも入れ替えるのも同じです。

初心者向きですが、上級者には火力不足感があるかもしれません。

#### ジェイドの小鳥

![ジェイドの小鳥](/img/game/hsr/origami-jade.png)

ブルーベリーを 5 個消すたびに、フィールド上の中央で 1 回、3×3 マスの爆発を起こす。

| 総合評価 | ★★☆☆☆ |
| -------- | ----- |
| 爆発力   | ★★☆☆☆ |
| 安定性   | ★★☆☆☆ |
| 簡単さ   | ★★★☆☆ |

爆発の位置が中央なので、フィールド効果で中央だけ点が増えるものを引けば高得点を狙えます。

画面端のフルーツやアイテムを消しづらく、盤面の回転率が悪いです。スーパー爆弾を縦に積む過程で中央の列に積まれてしまうと能力の爆発を阻害してしまうので、スーパー爆弾ループとの相性も悪く、得点が伸びづらいです。

ジェイドの小鳥を使う場合、左右端以外の列にはスーパー爆弾を積まないことが大切です。中央 3 列は爆発を阻害しますし、端から 2 列目は端の列のフルーツやアイテムを消しづらくなってしまいます。
また、ブルーベリー以外のフルーツ、特に左右や下段のフルーツを入れ替えることも大切です。「お掃除スプレー」を使ってもブルーベリー不足になりがちで、スーパー爆弾で全消しを狙いたいのですが……端に積めないと苦しい盤面になりやすく打開しづらいのが辛い性能です。

### 平和ボタン型

アイテム「平和ボタン」で爆弾が即爆発する状態を維持し続ける小鳥です。
「平和ボタン」がないと全く強くありませんが、「平和ボタン」さえあれば他の連爆型を上回る効率の連爆を連発できます。
少しでも消して盤面を入れ替え続け、「平和ボタン」を引くことが勝利の鍵です。
スーパー爆弾による全消しはできなくても点数が伸びやすいです。

連爆型同様、連鎖中は操作をゆっくり考えられるのでコントローラーの人でもやりやすいです。ただ「平和ボタン」の扱いに慣れていないと難しいので、初心者向きではないと思います。

#### 三月の小鳥

![三月の小鳥](/img/game/hsr/origami-march.png)

イチゴを 6 個消すたびに、ランダムなフルーツ 4 個を直線爆弾に変える。
直線爆弾は消される時、その爆弾のある行または列を爆破する。

| 総合評価 | ★★★☆☆ |
| -------- | ----- |
| 爆発力   | ★★★★☆ |
| 安定性   | ★★★☆☆ |
| 簡単さ   | ★★★★☆ |

イチゴを消し続けること、盤面を入れ替え続けて「平和ボタン」を引き続けることだけを考えれば良いです。

ロビンの小鳥より火力不足を感じやすいです。能力の回転率は良いですが、同じ行に横の直線爆弾、同じ列に縦の直線爆弾が配置され火力ロスになりやすいです。
また、直線の爆発は連鎖になりづらいので連鎖も一段で終わりがちです。

爆発範囲と回転率のカタログスペックは優秀なので、大連鎖を引き当てるまでイチゴを消し続けましょう。

ちなみに「平和ボタン」がないとガス欠になるので毎ラウンドで「平和ボタン」を引かなければなりません。運に左右されがちです。

#### ロビンの小鳥

![ロビンの小鳥](/img/game/hsr/origami-robin.png)

モモを 7 個消すたびに、ランダムなフルーツ 4 個を正方形爆弾に変える。
正方形爆弾が消される時、3×3 マスの爆発を起こす。

| 総合評価 | ★★★★☆ |
| -------- | ----- |
| 爆発力   | ★★★★★ |
| 安定性   | ★★★☆☆ |
| 簡単さ   | ★★★★☆ |

三月の小鳥同様、モモを消し続けることと「平和ボタン」を引き続けることだけを考えれば良いです。

爆発の形状が良く、連鎖を起こしやすいです。開拓の小鳥と似ていますが、こちらのほうが回転率が悪い代わりに爆発範囲が非常に大きいので、「平和ボタン」さえ引ければロビンのほうが点数が伸びやすいです。

「平和ボタン」を引き続ける運は必要ですが、最上位で戦えるだけの大連鎖で点数を稼ぎやすいです。個人的には銀狼、ブートヒルに迫るほどの高得点を狙えるポテンシャルがあると感じています。
